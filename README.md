# FlashQuiz - Interactive Quiz Platform

FlashQuiz is an engaging and interactive quiz platform built with **React.js**. It supports **Multiple Choice Questions (MCQ)** and **Numerical Answer Type (NAT)** questions, provides instant feedback, and tracks quiz attempts using **IndexedDB**.

---

## Features

 **MCQ & NAT Questions** ‚Äì Supports multiple-choice and numerical input questions.  
 **Instant Feedback** ‚Äì Users get immediate feedback after selecting an answer.  
 **Timer-based Quizzes** ‚Äì Each question has a **30-second** timer.  
 **Progress Tracking** ‚Äì Displays a progress bar to indicate quiz completion.  
 **Scoreboard & Medals** ‚Äì Awards **Gold, Silver, or Bronze** medals based on performance.  
 **Quiz History (IndexedDB)** ‚Äì Stores previous quiz attempts locally for review.  
 **Modern & Responsive UI** ‚Äì Optimized for both mobile and desktop users.  

---

## Tech Stack

- **Frontend:** React.js, TypeScript  
- **State Management:** React Hooks (`useState`, `useEffect`)  
- **Storage:** IndexedDB (for saving quiz history)  
- **Navigation:** React Router
